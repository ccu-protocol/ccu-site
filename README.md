# CCU Protocol

Verified Compensation Credits

CCU is a humanitarian compensation protocol based on a
Proof-of-Compensation model with a dual-emission structure.

The protocol is designed for transparent issuance, auditable recovery,
prevention programs, and emergency humanitarian support.

Network: Polygon Mainnet  
Token Symbol: CCU  
Decimals: 6

Official Website:
https://ccu-protocol.github.io/ccu-site/
