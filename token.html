Wallet Notes

  • Users should add the token contract address, not a wallet address.
  • Wallets need POL for transaction fees on Polygon.
  • Operations, treasury, and governance wallets should always keep gas reserves.
  • CCU uses 6 decimals.

Token Utility Context

CCU is intended as a verified compensation credit for policy-bounded issuance, transparent operations, emergency response, and auditable regeneration or prevention programs.

Important Distinctions

  • Token Contract Address: defines the token itself
  • Wallet Address: holds balances
  • Private Key / Seed: controls spending authority

Never enter a wallet address into the “Custom Token” field when you intend to add CCU to a wallet.

Users should always verify contract address, network, and decimals before adding any token to a wallet.
